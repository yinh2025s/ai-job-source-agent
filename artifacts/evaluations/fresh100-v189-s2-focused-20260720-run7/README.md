# Fresh 100 `.189` S2 Focused Artifact

This directory preserves the Phase C summary for the fixed 49-record S2 failure subset.
It is a focused diagnostic, not a replacement for the immutable `.188` fresh-100 cold
benchmark or closure matrix. The authoritative live and replay roots are named in
`evaluation-summary.json`; the release archive preserves their complete contents.

Files:

- `s2-closure-delta.csv`: record-by-record `.188` versus `.189` S2 outcome.
- `evaluation-summary.json`: aggregate transport, retry, boundary, identity, and replay gate.
- `manifest.json`: source and artifact digests.
